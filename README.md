# LayerZero × KelpDAO 事件剖析

單頁靜態簡報：以 HTML/CSS 呈現 LayerZero 與 KelpDAO 相關安全事件的技術整理與時間線。

## 本機預覽

簡報主檔為 `slides.html`，可用以下任一方式開啟：

### 方式一：直接以瀏覽器開啟

雙擊 `slides.html`，或於專案目錄執行：

```bash
open slides.html        # macOS
xdg-open slides.html    # Linux
start slides.html       # Windows
```

### 方式二：透過本機靜態伺服器（推薦）

在專案目錄啟動任一靜態伺服器，例如：

```bash
python3 -m http.server 8080
```

接著於瀏覽器開啟 `http://localhost:8080/slides.html`。

> 若使用其他工具（如 `npx serve`、`live-server`），同樣指向根目錄並訪問 `slides.html` 即可。
